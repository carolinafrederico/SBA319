Welcome to the Blog API Server

A MongoDB Database Application

🚀 Overview

This server provides full CRUD operations for Users, Posts, and Comments. It’s designed for developers to test, build, or integrate into blogging platforms or learning projects.

📡 Available Routes

👤 Users

  • GET /user - Fetch all users
  • GET /user/:id - Get a specific user
  • POST /user - Create a new user
  • PUT /user/:id - Update a user
  • DELETE /user/:id - Delete a user

📝 Posts

  • GET /post - List all posts
  • GET /post/:id - Get a single post
  • POST /post - Create a new post
  • PUT /post/:id - Update a post
  • DELETE /post/:id - Delete a post

💬 Comments

  • GET /comment - List all comments
  • GET /comment/:id - Get a comment by ID
  • POST /comment - Add a new comment
  • PUT /comment/:id - Edit a comment
  • DELETE /comment/:id - Delete a comment

🧪 How to Use

You can test the routes using Postman or curl. Example:

POST http://localhost:3004/user

Body (JSON):

{
  "firstName": "Test",
  "lastName": "User",
  "email": "test@example.com",
  "password": "secret",
  "dob": "1990-01-01",
  "post": "PostObjectIdHere",
  "comment": "CommentObjectIdHere"
}

Make sure to insert valid ObjectIds from existing posts and comments!

📘 What I've Learned

  • Use MongoDB Compass or Mongoose queries to browse data directly.
  • Make sure your MongoDB connection string is valid in .env.
  • Seed your database with seedData.js before testing.
